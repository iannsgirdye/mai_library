# Учебные материалы ФИИТа


## Предложить материал
Если ты хочешь помочь проекту с помощью добавления в него нового материала (ресурса или файла), достаточно написать об этом [владельцу проекта](https://t.me/iannsgirdye). Взамен ты будешь добавлен на доску почёта (подробно в `about/people`).


## Примечания
- У некоторых пользователей возникал один и тот же вопрос: **"Почему нельзя представить все файлы с помощью одной ссылки?"** Дело в том, что на диске хранятся файлы, которые не предназначены для общего доступа, поэтому пришлось представлять общедоступные файлы точечно, то есть через отдельные ссылки на группы файлов.
- В учебных ресурсах и файлах **mai_library** собрано только то, что пригодилось группе М8О-111БВ-24. Рекомендую при необходимости смотреть хранилища других ребят, так как программы обучения нередко меняются.
