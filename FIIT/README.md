# Учебные материалы ФИИТа


## Ответы на некоторые вопросы
— **Почему нельзя представить все файлы с помощью одной ссылки?** 
— Во-первых, на диске хранятся личные файлы [владельца проекта](https://t.me/iannsgirdye). Во-вторых, удобнее представить все материалы по каждому предмету в отдельных папках, поэтому файлы представлены точечно, то есть через отдельные ссылки на конкретные группы файлов.

— **Какие материалы представлены в mai_library?**
— В учебных материалах **mai_library** в основном собрано то, что пригодилось группе М8О-111БВ-24. Рекомендую при необходимости смотреть хранилища других ребят, так как программы обучения нередко меняются.
