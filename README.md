# mai_library
Сборник файлов и ресурсов, которые обязательно пригодятся студенту Московского авиационного института во время учебной и внеучебной деятельностей.
Всё представленное собрано владельцем репозитория, куратором его группы и его одногруппниками.

## Примечание
У некоторых пользователей возникал один и тот же вопрос: <b>"Почему нельзя представить все файлы с помощью одной ссылки?"</b><br>
Дело в том, что на диске хранятся файлы, которые не предназначены для общего доступа, поэтому пришлось представлять общедоступные файлы точечно, то есть через отдельные ссылки на группы файлов.
