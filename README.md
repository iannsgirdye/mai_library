# О проекте
Сборник файлов и ресурсов, которые обязательно пригодятся студенту [Московского авиационного института](https://mai.ru) во время учебной и внеучебной деятельностей. 

- [Учебные материалы ФИИТа](/FIIT/) — для избранных.
- [Справочники](/manuals/) — полезная информация по популярным темам.
- [Внутренности проекта](/project/) — доски почёта и версионирование.
- [Общие ресурсы](/resources.md) — всё то, что есть в МАИ.
- [Разное](something.md) — всё то, чему нигде не нашлось места на репозитории.


## Структура
- [Telegram-канал](https://t.me/mai_library) — ссылка на репозиторий, чтобы не терять его.
- Хранилище — сборник файлов, ссылки на которые представлены в предметных `README.md` (конкретно для ФИИТовцев).
- [Репозиторий](https://github.com/iannsgirdye/mai_library) — сборник ресурсов (ты прям здесь).
- [Фото-конспекты](https://t.me/mai_library_photo) — фотографии записей на досках (и это тоже для ФИИТовцев).


## Поддержать проект
Помочь проекту можно одним из способов (но это не ограниченный перечень):
- рассказать другим о нём;
- предложить новый материал (файл или ресурс);
- сообщить об ошибке, опечатке или неточности;
- предложить идею по улучшению;
- накормить [владельца проекта](https://t.me/iannsgirdye) в "Додо" или "Теремке".

Для этого достаточно написать об этом [владельцу проекта](https://t.me/iannsgirdye). Взамен ты будешь добавлен на соответствующую доску почёта (в [`/project/people`](/project/people/)).
