# mai_library
Сборник файлов и ресурсов, которые обязательно пригодятся студенту Московского авиационного института во время учебной и внеучебной деятельностей.
Всё представленное собрано студентами и куратором группы М8О-111БВ-24.

## Примечания
- У некоторых пользователей возникал один и тот же вопрос: **"Почему нельзя представить все файлы с помощью одной ссылки?"** Дело в том, что на диске хранятся файлы, которые не предназначены для общего доступа, поэтому пришлось представлять общедоступные файлы точечно, то есть через отдельные ссылки на группы файлов.
- В **mai_library** собраны только те файлы, которые пригодились группе М8О-111БВ-24 (в большинстве случаев). Рекомендую при необходимости смотреть хранилища других ребят (выше), так как программа обучения нередко меняется.
