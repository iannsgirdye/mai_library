# Версионирование


## 1.0.0 (2025.01.08)

### 1. Перенос проекта
Для повышения удобства использования проекта он был перенесёт с канала в Telegram на репозиторий в `GitHub`. <br>
Отсчёт версий будет вестись относительно обновлений на репозитории.

### 2. Обновление диска
Каждый предмет теперь разделён на 2 категории: "Задания" и "Материалы". В первой расположены файлы, которые необходимы для решения и выполнения всех возможных заданий; во второй — необходимая для всего теория.

### 3. Пополнение банка файлов и ресурсов
На диск и в репозиторий были загружены много новых актуальных файлов и ресурсов соответственно на любой вкус и цвет.

### 4. Добавление собственных конспектов владельца
Для личного удобства владельца он подключил систему контроля версий для конспектов, которые он самостоятельно создаёт, и заодно поделился с пользователями репозитория.

### 5. Подробная информация о каждом предмете
Теперь все предметы для удобства представлен в отдельных `README.md`. В добавок к этому было увеличено наполнение сведений по каждому предмету.

### Примечание
У некоторых пользователей возникал один и тот же вопрос: <b>"Почему нельзя представить все файлы с помощью одной ссылки?"</b> Дело в том, что на диске хранятся файлы, которые предназначены только для его владельца, поэтому пришлось представлять общедоступные файлы точечно, то есть через отдельные ссылки на группы файлов.
