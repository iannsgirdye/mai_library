# Доска почёта 
*aka Вклады людей в проект*


## Как попасть на неё?
Для того чтобы начать красоваться на ~~витрине как рыба~~ доске почёта, нужно помочь проекту одним из способов, указанных в главном `README.md` (главная страницы репозитория).


## Файл мой, на доске не я. Как так?
В основном владелец проекта загружает файлы и добавляет ресурсы, которые были отправлены в студенческие группы. Поэтому если возникла ситуация присвоения твоего файла другому человеку, обязательно напиши об этом [мне](https://t.me/iannsgirdye).


## Примечания
- Ресурсы и файлы, которые передаются всем студентам от преподавателей, не учитываются (за исключением нестандартной информации, то есть не распространяемой всем подряд).
- Вклад владельца проекта не учитывается (он колоссален), поэтому если предмет не упомянут на доске, значит никто, кроме владельца проекта, ничего не предложил.
