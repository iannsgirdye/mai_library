## Ключевое
- **Для кого:** ФИИТ и ПМИ.
- **Сроки:** с 30.06.2025 по 13.07.2025.
- **Прохождение:** в оффлайн режиме, поэтому уехать домой и проходить онлайн нельзя.
- **Отчётность:** после завершения практики необходимо подготовить отчёт по установленной форме и сдать его на кафедру 806 до 13.07.2025.
- **Спойлер:** в середине мая (после майских праздников) будет собрание по практике и отдельная беседа.



## Варианты мест прохождения
### Не для целевиков
- **Лаборатории кафедры (можно выбрать один из следующих треков)**:  
    - Искусственный интеллект;
    - Креативные индустрии (видеомонтаж);
    - Программное обеспечение для беспилотных летательных аппаратов (БПЛА);
    - Разработка программного обеспечения (Backend/Frontend).
- **Олимпиадное программирование**
- **Школа IT и математического моделирования:**
    - с 27.04.2025 по 05.05.2025;
    - [регистрация](https://matmod.itmai.ru).
- **Авиахакатон:** декабрь 2024.
- **Разметка данных (в области зондирования Земли):**
    - прохождение досрочно в течение текущего семестра;
    - заявки принимаются до 11.04.2025, по поводу записи необходимо написать [Полине](https://t.me/orlando_plbre).
- **В своей компании:**
    - необходимо самостоятельно заключить договор о практике до 12.06.2025;
    - шаблоны документов можно найти на [сайте МАИ](https://mai.ru) или в дирекции института.


### Для целевиков  
Прохождение практика на своих базовых предприятиях согласно договору.
