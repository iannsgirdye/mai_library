### Проект (М8О-111БВ-24)
- **Организационное:**
    - **Выбор:** вид проекта нужно выбрать самостоятельно, а его темы уже распределяет сам преподаватель.
    - **Состав:** в команде 3-4 человека из одной группы, между которыми нужно распределить роли.
    - **Критерии:** будут отправлены позже
    - **Выполнение:** в течение семестра
    - **Сдача:** в зачётную неделю
    - **Влияние:** допуск к коллоквиуму и учёт в итоговой оценке за дисциплину
- **Виды:**
    - **Проект №1 — "Парсинг и витрина данных":**
        - Backend: парсер (парсинг внешних источников данных)
        - Frontend: статистические данные
        - PostgreSQL
        - LLM
    - **Проект №2 — Аппроксимация траектории твёрдого тела:**
        - Backend: Kinematic approximation (ML)
        - Frontend: 3D моделирование
        - ...
    - **Проект №3 — Оболочка к высокопроизводительному процессу на C++ или Fortran:**
        - Backend: Python Wrapper
        - Frontend: PyGame, GYM
        - Что-то с GPU
