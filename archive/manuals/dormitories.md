# Справочник: Общежития
> [!NOTE]
> - [Структура руководства](#структура-руководства)
> - [Наказания](#наказания)



## Структура руководства
Дирекция студенческого городка → общежитие → начальник корпуса → комендант



## Наказания
- **От слабого к жёсткому:** замечание → выговор → выселение
- **Эквивалентность:** 3 замечания = выговор, 3 выговора = выселение
