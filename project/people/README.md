# Доска почёта 
*aka Вклады людей в проект*


## Как попасть на неё?
Для того чтобы начать красоваться на ~~витрине как рыба~~ доске почёта (а может и на нескольких сразу), достаточно помочь проекту одним из способов, указанных в главном `README.md` (главная страница репозитория).


## Файл мой, на доске не я. Как так?
В основном [владелец проекта](https://t.me/iannsgirdye) загружает те файлы и добавляет те ресурсы, которые были отправлены в студенческие группы. Поэтому если возникла ситуация присвоения твоего файла другому человеку, обязательно напиши об этом [мне](https://t.me/iannsgirdye).


## Примечания
- Не учитваются ресурсы и файлы, которые передаются всем студентам от преподавателей.
- Вклад [владельца проекта](https://t.me/iannsgirdye) не учитывается (он колоссален), поэтому некоторые предметы могут быть не упомянуты на доске.
